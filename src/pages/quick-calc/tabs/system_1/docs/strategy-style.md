## 一、风格定位

### 核心风格：关键位反转交易（Reversal at Key Levels）

在价格到达**重要支撑/阻力位**时，等待**反转确认信号**后入场，捕捉反转行情的一段利润。

不是猜顶/猜底 → 而是在关键位置，等市场先走出反转迹象，再跟进

### 为什么选这个风格

| 特征                                | 匹配度                      |
| ----------------------------------- | --------------------------- |
| 你赚钱的交易大多是逆向（抄底/做空） | 天然偏好                    |
| 你耐心好，能等两周                  | 反转机会不常出现，需要等待  |
| 你早晚看盘，不能盯盘                | 日线分析不需要频繁看盘      |
| 你用K线和技术指标                   | 反转信号可以用K线和指标量化 |

### 与你过去做法的关键区别

| 过去的做法                         | 新系统的做法                                  |
| ---------------------------------- | --------------------------------------------- |
| 看到大跌就觉得"该反弹了"，直接进场 | 必须在**关键位置** + 出现**确认信号**后才进场 |
| 凭感觉判断"跌够了"                 | 用客观标准定义"什么叫关键位"                  |
| 没有止损或止损太远                 | 止损紧贴关键位另一侧，幅度明确                |
| 方向看反了死扛                     | 止损触发就走，不犹豫                          |

## 二、基础参数

| 参数     | 设定        | 说明                                 |
| -------- | ----------- | ------------------------------------ |
| 时间框架 | 日线（1D）  | 早晚各看一次即可，不需要盯盘         |
| 辅助框架 | 4小时（4H） | 找更精确的入场点时参考               |
| 持仓周期 | 3~15天      | 典型的波段持仓时间                   |
| 交易方向 | 多空皆可    | 但做多和做空的确认标准不同（见下方） |
| 交易频率 | 每月 2~4 笔 | 宁可错过，不可做错。没有信号就空仓   |

## 三、品种范围

### 主要品种池

| 类别     | 币种                           | 交易频率                       |
| -------- | ------------------------------ | ------------------------------ |
| 核心品种 | BTC, ETH                       | 最常交易，信号最可靠           |
| 备选品种 | SOL, BNB, XRP, DOGE 等前20大币 | 有明确信号时交易               |
| 机会品种 | 其他山寨币                     | 仅在极端行情（暴跌>30%）时考虑 |

### 品种筛选原则

- **观察名单**：不设成交量门槛，任何币都可以画关键位和观察
- **入场门槛**：开仓当日 24h 成交量 ≥ **200万 USDT**（低于此数止损滑点风险大）
- **上线时间 > 6个月**：太新的币没有足够的历史价格数据做技术分析

## 四、什么是"关键位"

关键位是价格大概率会产生反应的位置。以下是本系统认可的关键位类型：

### 4.1 水平支撑/阻力位

- 前期明显的高点/低点
- 多次测试未突破的价位
- 长时间横盘的区间上下沿

**判定标准**：该价位至少被**2次以上**的K线高点或低点触及。

### 4.2 关键均线

- MA50（日线50日均线）
- MA200（日线200日均线）

在趋势中，价格经常在这些均线处获得支撑或阻力。

### 4.3 前期成交密集区

- 大量换手发生过的价位区间
- 在K线图上表现为长时间横盘整理的区域

## 五、什么是"确认信号"

到达关键位后，**不立即入场**，等待以下确认信号至少出现 1 种：

### 5.1 K线反转形态（日线级别）

做多确认：

- 锤子线 / 十字星（带长下影线，下影线长度 ≥ 实体 2 倍）
- 看涨吞没（阳线完全包住前一根阴线）
- 早晨之星（三根K线组合反转形态）

做空确认：

- 上吊线 / 射击之星（带长上影线）
- 看跌吞没（阴线完全包住前一根阳线）
- 黄昏之星

### 5.2 成交量确认

- 到达关键位时出现**放量**（当日成交量 > 5日均量的 1.5 倍）
- 放量方向与你预期的反转方向一致

### 5.3 指标背离（辅助参考）

- RSI 与价格背离：价格创新低但 RSI 未创新低（看多）
- RSI 与价格背离：价格创新高但 RSI 未创新高（看空）

## 六、风格执行纪律

### 必须遵守

1. 不在非关键位入场 — 价格在"半空中"时，不管它跌了多少，都不交易
2. 不在无确认信号时入场 — 到了关键位但没有反转K线，继续等
3. 不逆大趋势 — 日线级别明确的下降趋势中，不轻易抄底（除非出现极端超卖）
4. 每月交易不超过 6 笔 — 超过说明标准太松，需要收紧

### 空仓是最好的仓位

> 没有符合条件的机会时，空仓等待就是最好的操作。
> 市场每天都在，但你的资金是有限的。

---

## 七、风格总结

一句话描述：
在日线级别的关键支撑/阻力位，
等待K线反转确认信号出现后入场，
持仓3~15天，捕捉反转波段利润。

交易公式：
关键位 + 确认信号 + 风控规则 → 入场
缺任何一个 → 不交易
