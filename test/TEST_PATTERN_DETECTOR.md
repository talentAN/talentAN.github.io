# K线模式检测工具

## 快速开始

```bash
node test-pattern-detector.js
```

## 功能

- ✅ 识别摆动高点和摆动低点
- ✅ 识别成交密集区
- ✅ 识别支撑位和阻力位
- ✅ 识别6种K线模式（3种看涨 + 3种看跌）
- ✅ 综合判断入场条件

## 输入方式

### 1. 使用示例数据（推荐）

选择菜单选项 `1`，直接用IRYSUSDT的历史数据进行测试

### 2. 手动输入JSON数据

选择菜单选项 `2`，粘贴JSON格式的K线数据：

```json
[
  ["1766707200000", "0.03082", "0.0313", "0.02899", "0.02978", "16620770", "504222.90589"],
  ["1766793600000", "0.02978", "0.03187", "0.02963", "0.0317", "10744561", "331809.1591"],
  ["1766880000000", "0.0317", "0.03196", "0.03081", "0.03107", "5696297", "178882.77379"]
]
```

**K线数据格式：**

```
[时间戳(毫秒), 开, 高, 低, 收, 成交量, 成交额]
```

### 3. 读取JSON文件

选择菜单选项 `3`，输入文件路径（相对或绝对路径均可）

## K线模式识别

### 看涨模式 (Bullish)

- **锤子线** (Hammer): 下影线很长，实体很小，无上影线或很小
- **看涨吞没** (Bullish Engulfing): 第一根是看跌K线，第二根是看涨K线，完全吞没前一根
- **早晨之星** (Morning Star): 三根K线组合，形成底部转折信号

### 看跌模式 (Bearish)

- **射击之星** (Shooting Star): 上影线很长，实体很小，无下影线或很小
- **看跌吞没** (Bearish Engulfing): 第一根是看涨K线，第二根是看跌K线，完全吞没前一根
- **黄昏之星** (Evening Star): 三根K线组合，形成顶部转折信号

## 支撑/阻力位识别

### 两种方法

1. **摆动点法**: 前后各2根K线的高/低点都高/低于当前K线
2. **密集区法**: ≥10根K线组成，振幅≤8%的区间

### 识别规则

- 至少2个摆动点落在价格 ±1.5% 范围，且间隔≥5根K线
- 支撑位强度 = 相关摆动点数量 + 密集区持续时间

## 入场条件

✅ **同时满足**：

1. 当前价格在支撑位/阻力位附近 (±2%)
2. 最后3根K线出现对应方向的K线模式

## 故障排除

**错误：`Cannot find module`**

- 确保脚本在项目根目录运行

**数据为空**

- 检查JSON格式是否正确
- 确保至少有5根K线（3根用于检测模式 + 2根用于摆动点）

**没有识别出支撑/阻力位**

- K线数据可能太少或振幅太小
- 尝试增加K线数据量
